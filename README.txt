Legislator Look-Up
___________________

To install, place the entire leglookup folder in your modules directory.

Go to Administer -> Site configuration -> Legislator Look-Up. Save your Sunlight Labs and Google Maps API Keys. (Register keys here: http://services.sunlightlabs.com/api/register/ and http://www.google.com/apis/maps/)

Then go to Administer -> Site building -> Blocks, and enable the Legislator Look-Up block.
